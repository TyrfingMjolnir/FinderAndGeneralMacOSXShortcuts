## Apple XCode
### Inspector
| Shortcut                   | Action                | MacOS X Specific |
| -------------------------: | :-------------------- | :-----: |
| ⌥ ⌘ 0 | Toggle Inspector visibility |
| ⌥ ⌘ 1 | File Inspector |
| ⌥ ⌘ 2 | Quick Help |
| ⌥ ⌘ 3 | Identity Inspector |
| ⌥ ⌘ 4 | Attributes Inspector |
| ⌥ ⌘ 5 | Size Inspector |
| ⌥ ⌘ 6 | Connections Inspector |
| ⌥ ⌘ 7 | Bindings Inspector | * |
| ⌥ ⌘ 8 | View Effect Inspector | * |
